# Dynamsoft Code Parser Module

The Dynamsoft Code Parser module converts data strings, typically encrypted in barcodes and machine-readable zones, into human-readable information. To learn more, please refer to [DCV (Dynamsoft Capture Vision) architecture]({{ site.dcvb_js}}).

For details, please refer to [Dynamsoft Code Parser for Your Website - User Guide]({{ site.dcp_js }}user-guide/).
